Function

var pluralize = function(word, number) {
	if(number === 1) {
		return word;
	}
	else {
		return word + "s";
	}
};
var wordInput = $("section.present :text[name='word']").val();
var numberInput = parseInt($("section.present :text[name='number']").val(), 10);
var result = pluralize(wordInput, numberInput);
alert(result);

Review

Function

  • A reusable block of code
  • Stored in a variable (defined) and run later (called)
  • Parameters are inputs required by the function
  • The return value is the output of the function
  • Variables are not accessible outside the function
